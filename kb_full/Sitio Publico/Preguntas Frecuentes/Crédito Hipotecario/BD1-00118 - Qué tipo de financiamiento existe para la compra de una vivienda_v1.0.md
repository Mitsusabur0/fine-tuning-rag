# ¿Qué tipo de financiamiento existe para la compra de una vivienda?

Para comprar una vivienda nueva o usada, BancoEstado cuenta con los siguientes tipos de financiamiento:

- Crédito Hipotecario con y sin subsidio habitacional.

- Financiamiento para EcoVivienda con y sin subsidio habitacional.

- Crédito Hipotecario Universal.
