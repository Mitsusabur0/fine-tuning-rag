# Cuál es el costo de portarme a Banco Estado

El proceso de portabilidad financiera no tiene costo para el cliente.

Por otra parte, puede tener un costo para el cliente la contratación de un producto, por ejemplo, los créditos hipotecarios donde el cliente debe pagar los gastos operacionales que implica la tasación, estudio de título, Notaría y Conservador en caso que se requiera.
