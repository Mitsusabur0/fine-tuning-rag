# ¿Tuviste un ingreso extra? 5 consejos para no gastarlo todo de una

A veces recibimos un "extra" inesperado: un bono, una devolución de impuestos o incluso un regalo. Y aunque es tentador gastarlo rápido, este tipo de ingresos pueden ser una gran oportunidad para mejorar tus finanzas.

Aquí te damos algunas ideas simples para aprovechar al máximo ese dinero adicional:

1. Sepáralo antes de gastarloApenas te llegue ese ingreso, aparta una parte para ahorrar. Puede ser un 10%, 20% o más, según tu situación. Si esperas demasiado, es fácil que lo gastes sin darte cuenta.

2. Cubre deudas pequeñasSi tienes pagos pendientes o cuotas chicas que te están generando interés, considera usar parte del ingreso extra para saldarlas. ¡Eso te dará mayor tranquilidad a futuro!

3. Crea un fondo para imprevistosAunque no sea mucho, puedes comenzar un fondo de emergencia. Tener un respaldo te puede ayudar a enfrentar situaciones difíciles, sin recurrir a créditos o avances.

4. Usa productos de ahorro simplesUn depósito a plazo es una buena opción si no necesitas el dinero inmediatamente. Es fácil de contratar y te permite guardar tu dinero de manera segura, generando un pequeño interés a favor.

5. Déjate un “gusto consciente”¡Ahorrar también es disfrutar con equilibrio! Puedes destinar una parte para darte un “gustito”, pero de forma planificada. Así no sientes que estás “renunciando” a todo.

¡Recuerda: no importa si el monto es grande o pequeño! Lo importante es generar el hábito de tomar decisiones informadas y usar esos extras a tu favor.
